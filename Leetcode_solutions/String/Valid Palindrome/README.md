## 문제 개요
주어진 스트링이 팔린드롬 -> 맨앞글자와 맨뒷글자가 일치하는지 판별하는 문제.

## 접근법
먼저 주어진 스트링이 뒤죽박죽이라 regex를 통하여 string에서 alphanumeric만 남기고, 그리고 toLowerCase로 소문자화한다.

그뒤 split('').reverse().join('') 해서 앞뒤로 뒤집은뒤 alphanumeric/toLowerCase된 스트링과 비교하여 참거짓 반환

아니면 투포인터 로 스트링 시작과 끝부터 좁혀가나가는 방법이 있음. 투포인터로 구현 선호.

## 시간/공간복잡도 
메소드 접근
시간 O(N)
공간 O(N) split reverse join들이 새로운 문자열과 배열 생성하기 때문에 O(N)

투포인터 (문자열이 커질수록 이방식이 유리)
시간 O(N)
공간 O(1)
