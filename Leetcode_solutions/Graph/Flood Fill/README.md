## 문제 개요
시작점에서 연결된 같은 색상의 영역을 새로운 색상으로 모두 채우는 알고리즘
## 접근법
BFS(선호)

Queue 를 사용하여 너비 우선 탐색 진행
<ol>
시작 좌표를 큐에 넣고 방문 처리(색상변경)
큐가 빌때까지 while loop 돌며 현재 위치에서 상하좌우를 확인시킴.
범위 내에 있고 기존 색상과 같다면 큐에 삽입 후 색상 변경.
</ol>

DFS
<ol>
현재 노드의 색상을 변경
현재 노드의 상하좌우를 순차적으로 확인
범위 내에 있고 기존 생상과 같다면 즉시 재귀 호출(floodFill(next_x, next_y)
호출 스택을 통해 모든 연결된 칸을 방문 후 종료
</ol>

## 시간/공간복잡도 

BFS
시간 O(N * M)
공간 O(min(N, M))

DFS
시간 O(N * M)
공간 O(N * M)
